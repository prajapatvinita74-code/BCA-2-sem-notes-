C Programming Notes

Deep and detailed concepts for learning C programming with examples

1. Introduction to C

C is a general-purpose programming language developed in 1972 by Dennis Ritchie. It is widely used for system programming and embedded systems.

2. Basics

  • Data Types: int, float, double, char
  • Variables and Constants
  • Input/Output: scanf() and printf()
  • Operators: Arithmetic, Logical, Bitwise

3. Control Statements

  • Conditional Statements: if, else, switch
  • Loops: for, while, do-while

4. Functions

Functions help in modular programming.

  • Defining and Calling Functions
  • Function Parameters and Return Values
  • Recursion with examples

5. Arrays and Strings

  • One-dimensional and Multi-dimensional arrays
  • String handling functions: strcpy(), strlen(), strcmp()

6. Pointers

Pointers store memory addresses. Key points:

  • Pointer declaration and initialization
  • Pointer arithmetic
  • Pointers and arrays
  • Function pointers

7. Structures and Unions

  • Defining struct and union
  • Accessing members using . and ->
  • Nesting structures

8. File Handling

  • Opening and closing files using fopen() and fclose()
  • Reading and writing: fread(), fwrite(), fprintf(), fscanf()
  • Error handling in files

9. Example Program

#include <stdio.h>

int main() {
    int a, b, sum;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    sum = a + b;
    printf("Sum = %d\n", sum);
    return 0;
}
